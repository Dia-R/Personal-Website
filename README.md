# Personal Website

A personal portfolio website built with HTML, CSS, and JavaScript. This project is being developed iteratively as a learning experience in core web development fundamentals.

🌐 **Live site:** [dia-r.github.io/Personal-Website](https://dia-r.github.io/Personal-Website)

---

## About

This site serves as a portfolio to showcase my projects and experience. It is built without frameworks intentionally — the goal is to understand the fundamentals of the web before reaching for abstractions.

---

## Tech Stack

- **HTML5** — semantic structure and content
- **CSS3** — layout (Flexbox/Grid), responsive design, animations
- **JavaScript (vanilla)** — interactivity and DOM manipulation
- **GitHub Pages** — hosting and deployment

## Running Locally

No build tools or dependencies required.

```bash
# Clone the repository
git clone https://github.com/Dia-R/Personal-Website.git

# Open in your browser
open index.html
```

Or simply open `index.html` directly in any browser.

---

## Development Approach

This project follows a feature-branch workflow:

- All new features are developed on their own branch (e.g. `feature/navbar`)
- Changes are merged into `main` via pull requests
- Commits follow [Conventional Commits](https://www.conventionalcommits.org/) — e.g. `feat:`, `fix:`, `style:`, `docs:`
- Work items are tracked as [GitHub Issues](https://github.com/Dia-R/Personal-Website/issues)

---

## Deployment

The site is deployed automatically via **GitHub Pages** from the `main` branch. Every merge to `main` is reflected live within a few minutes.
